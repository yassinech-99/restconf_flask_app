Restconf-Flask

This is a simple app that utilizes Restconf API to retrieve information from the Cisco IOS XE datastore.

Feel free to explore and manage your devices using the provided features.

Add Devices Tab

The "Add Device" tab is used to create, update, and delete devices. also from the With Selected tab you can click:

Get Interface Information: to query device interfaces information.

Get Device Configuraton: to get the device configuration and save in a json format in the config-files directory.

Send Configuration Set:(comming soon) to upload configuration file in a json format used to make changes to the device datastore
